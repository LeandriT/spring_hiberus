# Carpeta Generations - Fragmentos Generados por IA

Esta carpeta contiene fragmentos relevantes de código generados por IA durante el desarrollo del proyecto.

## Propósito

- Almacenar ejemplos de código generado por IA
- Documentar artefactos importantes generados automáticamente
- Mantener un historial de generaciones relevantes para referencia

## Tipos de Archivos que Pueden Almacenarse Aquí

- Archivos OpenAPI/Swagger generados (`openapi.yaml`, `openapi.json`)
- Controladores generados desde contratos OpenAPI
- Mappers de MapStruct generados
- Otros artefactos generados por herramientas

## Estructura Sugerida

```
ai/generations/
├── openapi/
│   └── openapi.yaml (ejemplo)
├── controllers/
│   └── (fragmentos de controladores generados)
├── mappers/
│   └── (fragmentos de mappers generados)
└── README.md (este archivo)
```

## Uso

Cuando se genere código relevante por IA (especialmente a partir de contratos OpenAPI o herramientas automatizadas), guardar una copia o referencia aquí para documentación y referencia futura.

## Nota

Los archivos aquí almacenados son principalmente para documentación. El código fuente real debe estar en las ubicaciones correctas del proyecto según la arquitectura hexagonal.
