# Carpeta AI - Documentación de Uso de IA

Esta carpeta contiene documentación relacionada con el uso de herramientas de IA durante el desarrollo del proyecto.

## Contenido

- **decisions.md**: Registro de decisiones arquitectónicas y correcciones manuales aplicadas durante la implementación.
- **steps.md**: (Opcional) Documentación de los pasos de implementación seguidos.

## Uso

Al trabajar con código generado por IA (como Cursor Pro), documenta aquí:
- Correcciones manuales necesarias
- Decisiones arquitectónicas importantes
- Problemas encontrados y cómo se resolvieron
- Desviaciones del código generado inicialmente
